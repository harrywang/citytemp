# Cities

A simple web app to compare temperature, population, and location data for cities around the world.

## Features

- Compare up to 10 cities with interactive charts
- Temperature: Monthly temperature line chart with Celsius/Fahrenheit toggle
- Population: Bar chart comparing city populations
- Location: Scatter plot showing geographic coordinates
- Export each chart as high-quality PNG
- Mobile-friendly responsive design

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Recharts
- shadcn/ui

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
